
# All individual charts description of content 

Tabs on Left
1. Clinical Dashboard
2. Data for Improvement
3. About us

## Pain - individual chart contents:

* Unit top left
* Title top centre: Pain Scores
* Legend below title: VPS- verbal pain scale (hollow circle), CPOT- Critical Care Observation Tool (hollow diamond); None/mild (green filled data point); Moderate (amber filled data point); Severe/very severe (red filled data point); Missing (grey bar)
* x axis: time day, date of month, e.g. 14:00, Wed 28
* y axis: Title: Pain, depicting pain score range (none, mild, moderate, severe, very severe)
* Slider bar: time window (up to last 72 hours)
* Back button bottom left
